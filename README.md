## 🚀 About Me

<img align="right" alt="Coding" width="400" src="https://cdn.dribbble.com/users/1162077/screenshots/3848914/programmer.gif">

I'm a frontend developer who builds the frontend of **Ratio EMS** and **RatioSIM** energy software end to end. I focus on turning complex technical domains into intuitive, performant, and scalable interfaces.

I've worked on WebSocket-based real-time infrastructure, SCADA single-line diagram editors, isometric facility editors, drag-and-drop dashboard engines, and React Native mobile apps. My background in civil engineering and production management helps me grasp technical domains quickly.

I prioritize code quality with TypeScript strict typing, Playwright E2E tests, and Storybook. I deliver product-focused solutions to enterprise clients through multilingual experiences.

**Highlights**

- Built the frontend architecture and modules of **RatioSIM** and **Ratio EMS** energy software end to end
- Designed Web Worker-based WebSocket infrastructure, a SCADA single-line editor, and a Babylon.js-powered isometric facility editor
- Worked on a 400+ file RatioSIM interface, a customizable KPI dashboard, and a React Native mobile app
- Contributed to testing and documentation with Playwright, Cypress, ESLint, and Storybook
- 🐍 API and backend development with **Python / Django**
